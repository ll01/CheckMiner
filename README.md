# CheckMiner
Checks hashrate of your miner at antpool 


arguments

-hash int
        your suspected hash rate (default 20)
        
  -var int
        tolarable variance (default 50)
        
  -wallet string
        your Etherium Wallet (default "0xF4Ef9C5413d27466277dB2043C8169b193B7DBF1")
